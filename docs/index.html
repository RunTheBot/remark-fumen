fumen2Canvas

Draw a Fumen page to a canvas

fumen2Canvas(fumenPage: Page, tilesize: number, numrows: number, transparent: boolean): Canvas
Parameters
fumenPage (Page) Fumen page to draw
tilesize (number) Size of a tile
numrows (number) Number of rows to draw
transparent (boolean) Whether to draw transparently
Returns

fumen2GIF

Draw a Fumen page to a GIFencoder

fumen2GIF(fumenPages: Array<Page>, tilesize: number, numrows: number, start: number, end: number, transparent: boolean): GIFEncoder
Parameters
fumenPages (Array<Page>) Fumen pages to draw
tilesize (number) Size of a tile
numrows (number) Number of rows to draw
start (number) Page to start drawing from
end (number) Page to stop drawing at
transparent (boolean) Whether to draw transparently
Returns
GIFEncoder: GIFencoder with the Fumen pages drawn on it
